ติดตั้งแอปศาลเจ้า

เพิ่มแอปลงบนหน้าจอมือถือ เพื่อใช้งานได้สะดวก รวดเร็ว และเข้าถึงศาลเจ้าได้ทุกที่ทุกเวลา

เปิดผ่าน Chrome บน Android เพื่อให้ติดตั้งได้สะดวกที่สุด

หากปุ่มไม่เด้งหน้าติดตั้ง

  1. เปิดหน้านี้ผ่าน Chrome เท่านั้น
  2. รีเฟรช 1–2 ครั้ง แล้วรอสักครู่
  3. หากยังไม่ขึ้น ให้กดเมนู ⋮ มุมขวาบน
  4. เลือก Install app หรือ Add to Home screen
